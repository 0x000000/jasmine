jasmine-ruby
============

    git clone git://github.com/ragaskar/jasmine-ruby.git
    cd jasmine-ruby
    rake jeweler:build
    rake jeweler:install
    cd ..
    mkdir test-jasmine
    cd test-jasmine
    jasmine init
    rake jasmine

then

    http://localhost:8888

and go!
